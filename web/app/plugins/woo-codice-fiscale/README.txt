WOOCOMMERCE Codice Fiscale FREE (GRATIS) is designed for Italian marketplaces running woocommerce to add CF and P.IVA to orders at checkout.

== Description ==

A long-awaited FREE plugin for the Italian online stores running WOOCOMMERCE plugin, it simply adds one mandatory field ( 'Codice Fiscale') and one optional field ( 'Partita Iva').
Both these fields will be shown -to users in their WOOCOMMERCE profile page
-to customers before checkout and after checkout in the "thank you" page
-to shop managers  in the backoffice order-edit page.
-in the email messages sent to user and shop managers

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all, no configuration needed
